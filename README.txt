Card descriptions

All cards:
The first four arguments on each line are the name, expansion, type of card, number in supply, and cost.

Victory:
For victory cards, this is followed by the point value, and a boolean value for whether this card is a default card.

Treasure:
For treasure cards, this is followed by the monetary value, and a boolean value for whether this card is a default card.

Action:
For simple action cards, this is followed by the numbers of +cards, +actions, +buys, and +money. Then we put an attack value.
-1 = no attack
0 = all other players gain a curse
1 = all other players with more than 3 cards discard two of them
